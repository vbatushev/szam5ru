#### Немного полезных ссылок для тех, кто пишет правила:

* Помощь по Adobe InDesign, раздел [“Найти/Заменить”](https://help.adobe.com/ru_RU/indesign/cs/using/WSFB3603CC-8D84-48d8-9F77-F3E0644CB0B6a.html) (рус.)
* InDesign ExtendScript API (eng): для [CS6](https://www.indesignjs.de/extendscriptAPI/indesign8/), [другие версии InDesign, Illustrator и Photoshop](https://www.indesignjs.de/extendscriptAPI/)

#### И для тех, кто пишет скрипты сам:

* Отличный бесплатный мультиплатформенный редактор кода [Microsoft VSCode](https://code.visualstudio.com/)
* [Adobe ExtendScript Debugger Extension для Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=Adobe.extendscript-debug) позволяет производить отладку и компилирование скриптов в JSXBIN прямо в редакторе VSCode
* [Раздел форума RU|DTP](https://forum.rudtp.ru/forums/skripty-i-plaginy-indesign.28/), посвященный скриптам и плагинам для InDesign
